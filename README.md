# Linux Learning Journey
## Commands
### pwd : Shows the current working directory
pwd
### ls : Lists files and directories
ls
### ls -ltr : Lists files in detailed format sorted by time
ls -ltr
### ls -a : Shows hidden files and directories
ls -a
### mkdir : Creates a new directory
mkdir linux
### cd : Change the current directory
cd linux
### cd .. : Moves one directory back
cd ..
### touch : Creates a new empty file
touch day1.txt
### echo : Writes text into a file
echo "Hello Linux" >day1.txt
### cat : Displays the content of  file
cat day1.txt
### whoami : Shows the current logged-in user
whoami
### mv : Moves or rename files
mv file1 projects
### rm : Removes files
rm file2
### rmdir : Removes an empty directory
rmdir kubernetes
### adduser : Creates a new user 
adduser sinchana
### Hidden file creation
touch .secret
